## Getting Started

### Prerequisites

- [Git](https://git-scm.com/) installed on your machine
- [Node.js](https://nodejs.org/en/download/) installed on your machine
- Basic familiarity with command line

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd ReactPortfolioTemplate
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open in your browser at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Customization

To customize this portfolio for your own use:

1. **Update Personal Information**: Edit `src/info/Info.js` with your details
2. **Change Colors**: Modify the `colors` array in `Info.js`
3. **Add Projects**: Update the `portfolio` array in `Info.js`
4. **Update Images**: Replace images in `src/img/` folder
5. **Modify Content**: Update bio, skills, and social links in `Info.js`
